# Frost - Study Productivity Hub

Frost is a sleek, web-based productivity tracker designed to help students and professionals manage their tasks, track focus sessions, and monitor productivity over time. It uses modern glassmorphism design and responsive layouts to work beautifully on both desktop and mobile devices.

## Features

- **Task Management:** Add, complete, and delete tasks with deadlines. Overdue or near-deadline tasks are highlighted.
- **Focus Timer:** Track focused study or work sessions with start, pause, and reset functionality.
- **Statistics Dashboard:** Monitor completed tasks, focus time, and overall productivity trends.
- **About Us & Contact Pages:** Learn more about the purpose of Frost and get in touch for feedback.
- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
- **Glassmorphism UI:** Modern, translucent card-style interface.

## Pages

1. `index.html` – Home page with user name input and dashboard overview.
2. `tasks.html` – Task management page with deadlines and productivity insights.
3. `focus.html` – Focus timer page for tracking study/work sessions.
4. `stats.html` – Productivity statistics overview.
5. `about.html` – Information about Frost and the need for a productivity tracker.
6. `contact.html` – Page to get in touch with the creator.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your preferred browser.
3. Start adding tasks, tracking focus sessions, and monitoring your productivity.

## Technologies Used

- **HTML5**
- **CSS3** (including glassmorphism effects)
- **JavaScript** (all pages share a unified `script.js` for functionality)
- **Google Fonts** (`Playfair Display` for headings)
